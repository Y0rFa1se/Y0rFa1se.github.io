Fermat's theorem

Let ppp be a prime and p∤ap \nmid ap∤a.

ap−1≡1(modp)a^{p-1} \equiv 1 \pmod{p}ap−1≡1(modp).

Euler's phi(ϕ\phiϕ) function

ϕ(n)=\phi(n) =ϕ(n)= # of k<n s.t. gcd⁡(k,n)=1k < n \ s.t. \ \gcd(k, n) = 1k<n s.t. gcd(k,n)=1

Properties

(a) if ppp is a prime, then ϕ(p)=p−1\phi(p) = p-1ϕ(p)=p−1

(b) ϕ\phiϕ is multiplicative. i.e. if gcd⁡(m,n)=1\gcd(m, n) = 1gcd(m,n)=1 then, ϕ(mn)=ϕ(m)ϕ(n)\phi(mn) = \phi(m)\phi(n)ϕ(mn)=ϕ(m)ϕ(n)

Euler's theorem

If gcd⁡(a,n)=1\gcd(a, n) = 1gcd(a,n)=1, then aϕ(n)≡1(modn)a^{\phi(n)} \equiv 1 \pmod{n}aϕ(n)≡1(modn)

  2. n = pq
  3. 1 < k < phi(n), gcd(k, phi(n)) = 1
